# Pokémon Beryl

A ROM hack of Pokémon Emerald using the [**Pokemon Emerald Decompilation Project**](https://github.com/pret/pokeemerald).

Currently, the project builds the following ROM:

**pokeberyl.gba** `sha1sum: a4733c79431d9e695ad57e207ee49ca3f3974c75`

# Installation
Installation instructions can be found in [INSTALL.md](./INSTALL.md).

# Changes
Changes introduced to the game can be found in [config.h](include/config.h). If you also want to enable or disable features before compiling, this is where you can configure the game to do so.
